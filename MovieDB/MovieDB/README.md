# MovieDB iOS App

TMDb Movie Database iOS Application with SwiftUI

See [README.md](MovieDB/README.md) for complete documentation.

## Quick Start

1. Open `MovieDB.xcodeproj` in Xcode
2. Build and run on your device or simulator
3. Enjoy browsing movies!

## Features

- Popular movies grid
- Search functionality
- Movie details with trailers
- Favorites management
- Clean MVVM architecture
